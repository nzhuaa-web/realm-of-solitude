# 🏰 寂境王朝 / Realm of Solitude

**Nopoly风格轻量生存策略游戏**

> 在虚空边缘重建希望，体验极简策略与生存压力的完美结合

## 🎮 游戏特色

- **Nopoly极简风格** - 低多边形视觉设计，干净利落的UI界面
- **轻量策略** - 少而精的决策，每个选择都至关重要
- **生存压力** - 昼夜循环驱动，虚空生物不断侵袭
- **高代入感** - 孤独王朝重建的沉浸式体验

## 🎯 核心玩法

### 昼夜循环系统
- **白天**：建造、规划、升级领地
- **夜晚**：防御虚空生物入侵
- **目标**：生存10个昼夜循环

### 王令策略系统
开局选择3个王令，定义你的游戏风格：
- **经济优先**：农场收益+30%
- **骑士荣耀**：骑兵获得冲锋击退
- **钢铁防线**：城墙耐久+50%
- **夜行者**：英雄夜战获得额外伤害
- **灵光护主**：死亡一次免死

### 灵光能量系统
- 策略+战斗双重资源
- 消耗动画和不足提示
- 技能释放的核心条件

## 🚀 快速部署

### 方案一：GitHub Pages（推荐）
```bash
# 1. 创建GitHub仓库
# 2. 上传所有文件到main分支
# 3. 在仓库设置中启用GitHub Pages
# 4. 访问 https://[username].github.io/[repository-name]
```

### 方案二：Netlify（一键部署）
```bash
# 1. 访问 https://netlify.com
# 2. 拖拽项目文件夹到部署区域
# 3. 自动生成部署链接
# 4. 支持自定义域名和HTTPS
```

### 方案三：Vercel（快速部署）
```bash
# 1. 安装Vercel CLI: npm i -g vercel
# 2. 在项目目录执行: vercel
# 3. 按照提示完成部署
# 4. 自动生成生产环境链接
```

### 方案四：本地服务器
```bash
# 使用Python内置服务器
python3 -m http.server 8000

# 或使用Node.js serve
npx serve . -p 3000

# 访问 http://localhost:8000 或 http://localhost:3000
```

## 🎮 游戏操作指南

### 开始游戏
1. 选择3个王令定义策略方向
2. 点击"开始游戏"按钮
3. 进入第1天白天阶段

### 白天阶段（建造）
- **采集灵光**：点击"采集灵光"按钮获取能量
- **建造建筑**：使用右侧建造按钮
- **升级英雄**：提升英雄等级和能力
- **规划防线**：合理布局防御设施

### 夜晚阶段（生存）
- **使用技能**：右下角技能栏释放英雄技能
- **监控敌人**：注意屏幕边缘预警箭头
- **保护建筑**：建筑受攻击会显示红色警告
- **生存到天亮**：抵御所有敌人波次

### 英雄技能系统
- **基础攻击**：对单个敌人造成伤害
- **防御姿态**：提升防御力3回合
- **灵光打击**：范围攻击，消耗灵光能量

- **金币/木材**：建造和升级基础资源

### 敌人波次
- 第1-3天：引导期，轻压力
- 第4-6天：构建体系期，出现精英怪
- 第7-9天：高压，多路线攻击
- 第10天：Boss战，最终考验

### 技能冷却
- 基础技能：1秒冷却
- 防御技能：3秒冷却
- 大招技能：5秒冷却 + 灵光消耗

## 🚨 故障排除

### 常见问题

**Q: 游戏无法加载？**
A: 检查浏览器是否支持现代JavaScript特性，尝试刷新页面

**Q: 按钮点击无响应？**
A: 确保已选择3个王令并点击"开始游戏"按钮

**Q: 部署后显示空白页面？**
A: 检查部署配置是否正确，确保survival_game.html是入口文件

**Q: 移动端显示异常？**
A: 游戏针对桌面优化，移动端建议横屏游玩

### 调试模式
在浏览器控制台输入以下命令启用调试：
```javascript
localStorage.setItem('debug', 'true')
location.reload()
```
